Billboard Demo
A Work on OpenGL.

This is the work I've finished independently during Computer Graphics lesson.
I take this one because it basically consists of representative technics utilized for Digital Arts:

  • Skybox
  • Particle System as the trail of the ball
  • Terrian gened by 2D perlin Noise
  • Texture gened by 3D perlin Noise
  • VBO technique

Plus, here are some innovative sparks:

  • Wrapping the whole terrian into a planet like one.
  • Using about 144 colors on the particle system.
    Each ball's color varying from one to another.
  • The color of the particle system changing constantly.
    From light to saturated and then back to light.
  • The color of the planet keeping changing slightly also.
  • Setting different cameras to trace the ball.

Linking Game
Character Designed.

We had a class named Curriculum Design of Programming during the summer semester in the first college year.

The topic is to design a certain game from the given 6 candidates. So I choose the popular flash game named 'Hello Corn'. Player should link as much corn as possible to get higher score.

The picture left shows the charactor I designed instead of the corn.

  • Different kind of eye means different functions.
    Like eliminate a line, a row or adjacent corns.
  • Different color means different kind of corns.
    You can only link the corns with the same color.
  • The light warp around the corn means it is linked
  • The tied foot means the corn is frozen and cannot fall down when the corn below is eliminated.

Musicube
A LeapMotion based Music Game.

This piece of work is a group project on HCI class. We combined leapmotion with music game and tried to get a totally new expierence for traditional music game palyers.

Unfortunately, leapmotion has some defects on accuracy and smooth level. So the final presentation is not so well. As seen in demo.

I mostly took part in the UI design part and work on the animation, shader, icon design parts together with other group members.

I used 3D collision on UI interact design and special gesture recognition in flat UI interact.

As you can see, the loading of icon means gesture recognized. To prevent the wrong detection, I chose the way of recognizing the gesture continued for a long time.
